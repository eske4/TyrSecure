# eBPF-Anticheat

## Build Instructions

Build the project using:

```sh
make build
```


## How to run

```make run```

## Requirements

Ensure the following dependencies are installed:

- **Clang** – Required to compile eBPF programs
- **bpftool** – Skeleton generation and eBPF management
- **cmake** – Project configuration and build system
- **libbpf** – Interface for interacting with eBPF programs
- **Linux kernel with eBPF enabled** (recommended: 5.x+)
